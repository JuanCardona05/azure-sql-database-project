# ☁️ Azure SQL Database Project

## 📖 Descripción

Proyecto enfocado en la creación y configuración de una base de datos en la nube utilizando Microsoft Azure SQL Database.

Se realizó el proceso completo de despliegue, configuración y conexión a la base de datos desde herramientas externas.

---

## 🚀 Objetivo

Implementar una base de datos en la nube utilizando servicios PaaS de Azure, garantizando conectividad y funcionamiento correcto.

---

## 🛠️ Tecnologías utilizadas

* Microsoft Azure
* Azure SQL Database
* SQL Server
* Herramientas de conexión (Azure Data Studio / SQL Server Management Studio)

---

## ⚙️ Implementación

* Creación de instancia de Azure SQL Database
* Configuración de servidor SQL en Azure
* Definición de credenciales de acceso
* Configuración de reglas de firewall
* Conexión a la base de datos desde cliente externo
* Validación de conexión y consultas

---

## 📂 Evidencia

Este repositorio incluye:

* Documento PDF con el proceso completo
* Capturas de la creación de la base de datos
* Evidencia de conexión exitosa

---

## 📚 Aprendizaje

* Uso de servicios PaaS en Azure
* Configuración de bases de datos en la nube
* Conectividad segura a servicios remotos
* Gestión básica de bases de datos SQL

---

## 🎯 Conclusión

Este proyecto permitió comprender el funcionamiento de bases de datos en la nube mediante Azure SQL, facilitando la administración y escalabilidad sin necesidad de gestionar infraestructura física.
